# Portfolio Website

This is a simple, modern, responsive portfolio website that you can use as an online version of your resume.

## Files

- `index.html` – main single-page portfolio layout (About, Skills, Experience, Projects, Education, Contact).
- `styles.css` – modern dark theme styling, fully responsive from mobile to desktop.
- `script.js` – small enhancements: mobile navigation toggle, smooth scrolling, active section highlighting, dynamic year in footer.

## How to use

1. **Open locally**
   - Open `index.html` directly in your browser, or
   - From this folder, run a simple server, for example:
     - With Python: `python -m http.server 8000` and then visit `http://localhost:8000`.

2. **Customize your content**
   - In `index.html`, replace all placeholder text:
     - `Your Name`, `Your Role / Title`, city, email, phone.
     - Experience, projects, and education entries.
   - Update the social links and project links to your real URLs.

3. **Adjust styling (optional)**
   - Tweak colors, spacing, or typography in `styles.css` as needed.

4. **Deploy online**
   - You can host this on any static hosting service (GitHub Pages, Netlify, Vercel, etc.) by uploading these three files.
